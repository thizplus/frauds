# แผนแก้ไข C01 — แจ้งโกง (Report Fraud)

> สร้าง: 30 พ.ค. 2569

## ปัญหาปัจจุบัน
1. กรอกข้อมูลไม่ครบทุกช่อง
2. ไม่กดปุ่ม "ส่งรายงาน"
3. ไม่รอหน้าสำเร็จ
4. Bank dropdown selector ใช้ try-catch skip

## แผนแก้ไข C01-report-fraud.spec.ts

| หลักฐาน | evidence1.jpg + evidence2.jpg | 2 รูป |

### Flow ใหม่

```
1. เปิดหน้าแรก → burger menu → "แจ้งข้อมูล"
2. เลือกประเภท "โกงเงิน"
3. กรอกชื่อ → นามสกุล → เลขบัตร → เบอร์โทร
4. scroll ลง
5. กรอกเลขบัญชี → เลือกธนาคาร (bank-dropdown-trigger → item)
6. scroll ลง
7. กรอก Social → กดปุ่ม + เพิ่ม
8. กรอกเล่าเหตุการณ์
9. scroll ลง
10. แนบหลักฐาน 2 รูป
11. scroll ลง → กดปุ่ม "ส่งรายงาน"
12. รอหน้าสำเร็จ "แจ้งรายงานสำเร็จ"
13. หน่วงดูผลสำเร็จ
```

### สิ่งที่เปลี่ยน

1. **ข้อมูลใหม่** — ชื่อ/นามสกุล/เรื่องเล่า ใกล้เคียงจริง
2. **Bank selector** — ใช้ `button.bank-dropdown-trigger` ตรงๆ (ไม่ try-catch skip)
3. **แนบ 2 รูป** — evidence1.jpg + evidence2.jpg
4. **กดส่งรายงาน** — `button[type="submit"]` ที่มี text "ส่งรายงาน"
5. **รอสำเร็จ** — waitForSelector `text=แจ้งรายงานสำเร็จ`
6. **ลบ try-catch ทิ้ง** — ทุก step ต้องทำงานจริง ถ้า fail ให้ test fail
