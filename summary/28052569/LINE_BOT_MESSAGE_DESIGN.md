---

## 1. กดปุ่มค้นหา (Search Prompt)

```
🔍 /AI-SEARCH
━━━━━━━━━━━━━
คุณกำลังเข้าสู่โหมดค้นหา

กรุณาพิมพ์ข้อมูลที่ต้องการตรวจสอบ:
▸ 📱 เบอร์โทร เช่น 0891234567
▸ 🏦 เลขบัญชี เช่น 1234567890
▸ 🪪 เลขบัตรประชาชน 13 หลัก
▸ 👤 ชื่อ-นามสกุล เช่น สมศักดิ์

📊 เหลือ 4/5 ครั้งวันนี้
⏱️ หมดเวลาใน 60 วินาที
```

สำหรับ member:
```
📊 👑 สมาชิก — ค้นหาไม่จำกัด
```

## 7. Welcome (แอดเพื่อนครั้งแรก)

```

━━━━━━━━━━━━━
ตรวจสอบประวัติคนโกงได้ทันที

วิธีใช้:
1️⃣ กดปุ่ม 🔍 ค้นหา ที่เมนูด้านล่าง
2️⃣ พิมพ์เบอร์โทร ชื่อ หรือเลขบัญชี
3️⃣ รอผลลัพธ์ไม่กี่วินาที

📊 ค้นหาฟรี 5 ครั้ง/วัน

```
