# LINE Bot Message Design — Text + Unicode Emoji

> ไม่ใช้ Flex Message — ใช้ text ธรรมดา + emoji + \n
> URL ใช้ LIFF URL (ไม่ใช่ /search?q= เพราะเว็บไม่รองรับ query param)

---

## หมายเหตุ
- LINE emoji (productId/emojiId) ใช้ได้แต่ซับซ้อน → ใช้ Unicode emoji แทน
- URL ทั้งหมดใช้ LIFF URL: `https://liff.line.me/2010174410-8ZWlb9uS`
- ข้อมูลสำคัญเน้นด้วย「」
- Section แยกด้วย ━━━ และ ───

---

```
━━━━━━━━━━━━━
คุณกำลังเข้าสู่โหมดค้นหา

▸ เบอร์โทร เช่น 0891234567
▸ เลขบัญชี เช่น 1234567890
▸ เลขบัตรประชาชน 13 หลัก
▸ ชื่อ-นามสกุล เช่น สมศักดิ์

📊 เหลือ 4/5 ครั้งวันนี้
⏱ หมดเวลาใน 60 วินาที

```
